# 实验规划

## 核心思路

基于 SD 预训练扩散模型，外挂轻量调制模块，实现**单步多退化图像恢复**。两阶段：

- **Stage 1** — 训练多退化分类器：DINOv2 backbone + Classifier Layer，二者均可训。输出 $\mathrm{logits} \in \mathbb{R}^{B \times C \times 2}$，逐行 softmax 取第一列得 $V \in \mathbb{R}^{B \times C}$
- **Stage 2** — 冻结 SD 主干，仅训练外挂调制模块 + 可选 LoRA。退化特征 $F_{\mathrm{Deg}} = U^T V$，$U \in \mathbb{R}^{C \times d}$ 为可训练字典。固定 timestep 加噪 → 单步去噪 → 输出恢复结果

参考：S3Diff ([2409.17058](https://arxiv.org/abs/2409.17058))、CODSR ([2512.14061](https://arxiv.org/abs/2512.14061))

---

## 实验设计总则

**影响力从大到小逐层贪心调优**：固定下层默认 → 测试上层选项 → 选优胜 → 继承 → 继续。

1. 固定 2,3,4,5 默认 → 测 1a / 1a-v3 / 1b → 选优胜
2. 固定 1（优胜）,3,4,5 默认 → 测 2a~2e → 选优胜
3. 固定 1,2（优胜）,4,5 默认 → 测 3a vs 3b → 选优胜
4. 以此类推

## 1. 调制模块设计

| 选项 | 名称 | 描述 |
|:---:|------|------|
| 1a | Cross-Attention 调制 | $F_{\mathrm{Deg}}$ 作 K/V(单token)，feature pyramid 作 Q，CrossAttention → $\alpha, \beta$，Conv 之后调制 |
| 1a-v3 | ResBlock 内 CrossAttn | $F_{\mathrm{Deg}}$ → MLP → 8 个语义 token 作 K/V，ResBlock 内 h₂ 作 Q（对齐 StableSR），GroupNorm 后调制 |
| 1b | CODSR LQFM 调制 | LQ → pixel_unshuffle → shared encoder → per-layer SFT → $\alpha, \beta$（time-aware $\lambda_t$）|
